
django-spotnet 0.1 documentation
================================

This documentation covers the 0.1 release of django-spotnet, a
simple pluggable application to manage spotnet posts
for `Django <http://www.djangoproject.com>`_-powered
websites.

This app includes all you need to get started with spotnet.
It everything handles retrieving posts from your NNTP server,
to pushing them to your download server (eg: Sabnzbd+ or NZBGet).

To get up and running quickly, consult the :ref:`quick-start guide
<quickstart>`, which describes all the necessary steps to install
django-spotnet and start using spotnet. For
more detailed information read through
the documentation listed below.
