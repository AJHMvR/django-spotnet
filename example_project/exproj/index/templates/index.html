Test project for django-spotnet

This is a very minimal application that shows the most default setup of django-spotnet.

  • The django builtin admin
  • The spotnet pages

To customize this app to your specific needs, please visit the documentation for all the options available.

  • Restyling any part by creating templates like 'spotnet/base.html', or 'spotnet/list.html" for more specific needs, in a place that takes precedence over app templates.
